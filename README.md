# Flat Capacitor simulation in COMSOL
Simple COMSOL simulation to calculate capacitance of a flat capacitor filled with air.
## Geometry
![Geometry](https://github.com/arizanloo/Flat-Capacitor-COMSOL/blob/main/Geometry.png)
## Results
Distribution of electric potential and electric field arrows inside the capacitor.
![Results](https://github.com/arizanloo/Flat-Capacitor-COMSOL/blob/main/Results.png)

Maxwell capacitance is 88.542 pF.
## Software
This project was created using COMSOL Multiphysics 6.3 .
